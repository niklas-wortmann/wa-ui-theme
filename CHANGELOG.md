# Changelog

All notable changes to this project will be documented in this file.

## [1.0.3] - 2026-03-26

- Fixed invalid `until-build` generation that blocked plugin structure validation.
- Aligned Java source/target/toolchain to Java 21 for IntelliJ 2025.3 compatibility.
- Upgraded IntelliJ Platform Gradle Plugin to 2.13.1.
- Added plugin metadata polish: vendor contact, richer description, and change notes.
- Added plugin icon at `src/main/resources/META-INF/pluginIcon.svg`.
- Updated all theme JSON `author` fields to `Jan-Niklas Wortmann`.
- Renamed theme variants for clarity: `Usable` -> `Balanced`, `Too Much` -> `Vibrant`.
- Added release documents: `README.md`, `LICENSE`, and this changelog.
